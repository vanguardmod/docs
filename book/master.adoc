[preface]
== Preface

This is the official manual for {vg-name} v{vg-version}.

{vg-name} is a competitive-focused modification for {et-name} ({et-name-abbr}),
built on the {etl-name} runtime. It targets ESL cup play and modern public
servers with a strong, military-toned identity.

This manual is the canonical reference for server administrators, competitive
players, and contributors. It is updated alongside every mod release and is
shipped both as a standalone GitHub Release at {vg-docs-repo} and bundled
inside every server ZIP from {vg-repo}.

[NOTE]
====
{vg-name} is currently in **Alpha** (v0.x.x). Cvars, commands, and behaviours
documented here may change between alpha releases. v1.0.0 will mark the
**Beta** milestone with a stable cvar contract.
====
