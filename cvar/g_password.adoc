[[cvar.g_password]]
=== g_password

[cols="1h,4",frame=all,grid=rows]
|===
| Default       | `"none"`
| Type          | string
| Flags         | `CVAR_USERINFO`
| Origin        | Game ({etl-name})
| In cup mode   | tunable
| Scope         | server
|===

Password required to connect to the server at all (i.e. to take *any*
slot, public or private). The literal default value `"none"` is treated
as "no password required" — if you want a join-password server, set
`g_password` to something other than `none`.

Distinct from <<cvar.sv_privatePassword>>: `g_password` controls the
gate to the server itself; `sv_privatePassword` controls whether you
land in a reserved slot.

[CAUTION]
====
Empty-string semantics differ between {et-name} mods. In stock {etl-name}
the literal string `none` means "no password"; setting `g_password ""`
also disables the password gate, but the canonical "off" value is
`none`.

Never commit a real password to a public Git repository.
====

==== Examples

.Open public server (no join password)
[source]
----
set g_password "none"
----

.Invite-only server
[source]
----
set g_password "CHANGEME-join"
----
